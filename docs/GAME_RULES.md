# Regras do Jogo

## Fluxo

1. O jogador cria um time.
2. Escolhe dificuldade, formação e estilo.
3. Faz draft de 11 titulares.
4. Joga a temporada 2026.
5. Se vencer a Libertadores, desbloqueia o Mundial.

## Overall

Cada jogador tem overall e atributos estimados pelo jogo:

- ritmo
- finalizacao
- passe
- drible
- defesa
- fisico

O overall do time é calculado por linha: goleiro, defesa, meio e ataque. A formação define onde cada jogador entra. O estilo de jogo adiciona bônus.

## Dificuldade

- `Difícil`: adversários recebem bônus pequeno.
- `Desafiador`: adversários recebem bônus maior e mata-mata fica mais duro.

## Velocidades

- `Normal`: eventos aparecem em ritmo de acompanhamento.
- `Rápida`: eventos aparecem mais rápido.
- `Pula final`: simula tudo até o fim da temporada.

## Draft

Cada rodada sorteia um time lendário. O jogador escolhe apenas um atleta desse time para encaixar na formação.

Se nenhum atleta agradar, é possível girar novamente até três vezes naquela escolha. Posições já preenchidas ficam bloqueadas, então não dá para escolher outro goleiro se o slot de goleiro já estiver ocupado.
